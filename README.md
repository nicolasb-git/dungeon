# Shadows of the Dungeon

A local browser-based graphical dungeon crawler (Dungeon). Explore deep into the shadows, battle monsters, and power up your hero in this grid-based adventure.

## Features

- **Character Selection**: Choose between three distinct classes: Warrior, Thief, and Sorcerer.
- **Dynamic Combat**: Turn-based combat where movement and positioning matter.
- **Deep Progression**: Monsters scale in difficulty as you descend deeper into the dungeon.
- **Equipment & Items**: Find gear, gold, potions, and food to survive and grow stronger.
- **Status Effects**: Experience unique mechanics like Paralysis and Pox debuffs from specific monsters.
- **Responsive UI**: A modern, immersive interface built with Cinzel and Outfit typography.

## Character Classes

| Class | Description | Key Strengths |
| :--- | :--- | :--- |
| **Warrior** | A battle-hardened fighter. | High Life, Good Sustain |
| **Thief** | A cunning and swift rogue. | High Power Growth, Stamina Efficient |
| **Sorcerer** | A powerful master of arcane forces. | Massive Power, Glass Cannon |

## Monsters & Hazards

- **Rat**: Fast and nimble, always wins initiative. Watch out for its **Pox** debuff!
- **Blob**: Slow but steady. Its touch can **Paralyze** you for a turn.
- **Skeleton**: A sturdy undead warrior with significant power.
- **And more...**: Spiders, Zombies, and Deamons haunt the lower depths.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (for running the local development server)

### Installation

1. Clone the repository to your local machine.
2. Navigate to the project directory.

### Running the Game

1. Start the development server:
   ```bash
   npm start
   ```
2. Open your browser to the URL provided by the terminal (typically `http://localhost:3000` or similar).

## Controls

- **Arrow Keys**: Move your character around the grid.
- **Walk into Monsters**: Initiate an attack.
- **Enter the Dungeon**: Start your journey from the character selection screen.
- **Reset Game**: Start fresh from Depth 1.

## Technical Stack

- **Language**: Vanilla JavaScript (ES Modules)
- **Styling**: Modern CSS3 with custom font integrations.
- **Server**: Hosted locally via `npx serve`.
